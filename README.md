# This will be our education plan

## Git

• Clone repository
• Create branches
• Make commits
• Push changes
• Pull changes
• Create & merge pull requests
• Rebase branches

## Javascript

1. Language Fundamentals
   • Variables (let, const, and var)
   • Data types (falsy)
   • Operators
   • Conditionals (if, switch)
   • Loops (for, while, for...of, for...in)

2. Functions
   • Function declaration vs function expression
   • Parameters and arguments
   • Arrow functions
   • Scope and hoisting
   • Closures
   • Pure and impure functions
   • Recursion

3. Objects and Arrays
   • Object literals
   • Methods
   • Destructuring
   • Spread and Rest
   • Array methods (map, filter, reduce, sort, find, some, every, flat, forEach, includes, indexOf, join, slice, splice, shift, unshift, pop, push, concat, etc.)
